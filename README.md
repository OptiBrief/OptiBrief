# OptiBrief — AI Market Intelligence

> *"The market compressed into only what matters to you."*

A YC-demo-ready AI market intelligence platform. Personalized, premium, screenshot-worthy.

---

## Run in 3 steps

```bash
# Windows
Expand-Archive optibrief-yc.zip -DestinationPath optibrief-yc
cd optibrief-yc
npm install
npm run dev
```

Open **http://localhost:3000**

---

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or: Push to GitHub → vercel.com → Import → Deploy (one click, no env vars needed).

---

## Features

| Section | What it does |
|---------|-------------|
| **My Feed** | Personalized intelligence feed — every card explains why YOU are seeing it |
| **Daily Brief** | Morning briefing — "the only 5 things that matter today" |
| **Narratives** | Market themes tracked by momentum, institutional attention, and acceleration |
| **Explain Move** | Click any stock → instant AI explanation of why it moved |
| **Watchlist** | AI-monitored assets with smart alerts and narrative tags |
| **Market Pulse** | Global sentiment, fear/greed, volatility — no charts, pure clarity |

---

## Design Philosophy

- **Whitespace is a feature** — every screen breathes
- **Personalization is visible** — every card shows "why you're seeing this"
- **Confidence is honest** — conflicting signals always surfaced
- **AI is invisible** — feels like intelligence, not a chatbot
